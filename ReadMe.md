# Harmony Reservations

A full-stack hotel reservation website built with the MERN stack.

## Features

- Browse available hotels
- Make and manage reservations
- User authentication
- Booking management

## Tech Stack

**Frontend:**
- React.js
- Vite

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

## Requirements

- Node.js (required)
- npm or yarn
- MongoDB

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables in both client and server folders

## Running the App

Start the server:
```bash
cd server
npm start
```

Start the client in another terminal:
```bash
cd client
npm run dev
```

## Project Structure

```
harmony-reservations/
├── client/    # React frontend
└── server/    # Express backend
```
